Analyze Adult Census Income. Original data is hosted on https://www.kaggle.com/uciml/adult-census-income

1. adult-income.R: This script trains prediction models and makes final prediction using random forest. It creates a "rdas" directory to save training results for rmarkdown, running this script requires: 
- linux file system
- the directory where the script runs writable

2. adult-income.pdf: the project report created by adult-income.Rmd

3. adult-income.Rmd: R markdown file used to create project report in pdf
